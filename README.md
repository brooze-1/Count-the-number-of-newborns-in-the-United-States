# Count-the-number-of-newborns-in-the-United-States
统计美国新生儿个数（python高级编程设）
《Python程序高级编程》课程设计任务：
1、完成特定网站上，特定数据的抓取；
2、将数据存到mysql数据库中；
3、读取数据库中的数据，构建成dataframe，并转换成特定的透视表；
4、基于构建出的透视表，绘制出特定图形；
5、以上任务都实现界面化。
6、实现界面上各个窗体的比例放缩。
